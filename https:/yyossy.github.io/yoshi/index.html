ファイザーワクチン

ファイザーとは

　ファイザーとは、ファイザーワクチンを開発した会社である。ファイザーワクチンを開発したのは現在アルバート・ブーラがCEOを務めている「ファイザー社」であり、事業内容としては、主に医療用薬品の製造・販売・輸出入を行っている。 　
　新型コロナのワクチンでも日本人の忌避の傾向は顕著である。世界経済フォーラムと調査会社イプソスが共同で15カ国を対象に実施した意識調査では、日本の新型コロナワクチンの接種意向は69％にとどまり、インドの87％や英国の79％よりも低く全体の平均の73％も下回った。 　

ファイザーワクチン

　ファイザーワクチンはメッセンジャー(mRNA)ワクチンである。
本剤を摂取し、mRNAがヒトの細胞内に取り込まれると、mRNAを基にスパイクタンパク質が生産され、それに対する中和抗体産生及び細胞性免疫応答が誘導されることでSARS-CoV-2による感染症の予防ができると考えられている。 ファイザーワクチンは新型コロナウイルス感染症の発症を予防する．
また，ファイザーワクチンはメッセンジャーmRNAワクチンワクチンである．厚生労働省(2021)によると，ファイザーワクチンを接種し，mRNAがヒトの細胞内に取り込まれると，このmRNAを基に細胞ｎ内でウイルスのスパイクタンパク質が生産され，スパイクタンパク質に対する中和抗体生産及び細胞性免疫応答が誘導されることで，SARS-CoV-2による感染症の予防ができると考えられるという．
厚生労働省(2021)によると，ワクチンを受けた人が受けてない人よりも，新型コロナウイルス感染症を発症した人が少ないということが分かっており，発症予防効果は約95%と報告されているという．
種類 mRNA
対象者 12歳以上
接種方法
間隔 3週間
有効性(発症予防効果) 約95%
副反応 頭痛、筋肉の痛み、疲労、寒気、発熱
対する説明など入ったらいいと思います 。ファイザーワクチンは2回接種する必要があるため、接種しに行くだけでも煩わしい。また、接種対象者は12歳以上に指定されている。
